# Inumaki Agent Instructions

These instructions apply to this top-level T3 project and all nested repos under it.

## Project Memory

- `.camie/project-memory.md` is the canonical durable memory for this project.
- Read `.camie/project-memory.md` at the beginning of every Codex session before making project-specific assumptions.
- At the end of each substantive turn, update `.camie/project-memory.md` with durable facts learned during the turn.
- Do not read or update a sibling project's `.camie` file for Inumaki work.
- Do not store secrets, tokens, passwords, service-role keys, temporary debugging notes, or short-lived task status in memory.
